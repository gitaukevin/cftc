# COT Analysis Dashboard

A comprehensive dashboard for analyzing CFTC Commitments of Traders (COT) data for Leveraged Funds and Non-Commercials.

## Features

- **Leveraged Funds Report**: Tracks Hedge Funds, CTAs, and Large Speculators
- **Non-Commercials Report**: Tracks Non-Commercial trader positioning
- **Market Selector**: Click buttons to view any market's historical data
- **Detailed Tables**: Shows Long, Short, Total OI, Percentages, Net Positions, and Weekly Changes
- **Line Charts**: Visualizes Net Positions over time
- **Trade Recommendations**: Identifies high-probability trading opportunities
- **Superior Trades**: Momentum-confirmed setups with accelerating bullish/bearish flow
